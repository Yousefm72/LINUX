Boot Process Overview

תהליך האתחול מתחיל ב-BIOS/UEFI, עובר דרך ה-bootloader, טוען את הקרנל ומאתחל את init/systemd.

dmesg מציג הודעות האתחול של הקרנל (kernel ring buffer).
dmesg | less מציג את הודעות האתחול בגלילה.
journalctl -b מציג לוג האתחול הנוכחי (עם systemd).
runlevel מציג את רמת הריצה הנוכחית (במערכות SysV init).
systemctl get-default מציג את ה-target ברירת המחדל (עם systemd).

Boot Sequence Simplified

תהליך האתחול במבט פשוט – חמישה שלבים עיקריים:

  1. BIOS/UEFI – בודק חומרה, מחפש התקן אתחול (דיסק, USB).
  2. Bootloader (GRUB) – טוען את קרנל Linux לזיכרון ומעביר לו פרמטרים.
  3. Kernel – מאתחל חומרה, טוען drivers, מעמיד מערכת קבצים root.
  4. init / systemd – תהליך האב הראשון (PID 1), מפעיל שירותים.
  5. Login – המערכת מוכנה – getty מציג מסך כניסה.

בקיצור: BIOS → GRUB → Kernel → systemd → Login

GRUB

GRUB (Grand Unified Bootloader) הוא bootloader הנפוץ ב-Linux. טוען את הקרנל ומעביר לו פרמטרים.

grub-install /dev/sda מתקין את GRUB על הדיסק (MBR או partition).
grub-install --target=i386-pc /dev/sda התקנה ל-BIOS (מערכת ישנה).
grub-install --target=x86_64-efi --efi-directory=/boot/efi התקנה ל-UEFI.
grub-mkconfig -o /boot/grub/grub.cfg יוצר קובץ הגדרות GRUB מחדש (GRUB2).

GRUB version 1

GRUB Legacy – גרסה ישנה. קובץ ההגדרה menu.lst נמצא ב-/boot/grub/.

/boot/grub/menu.lst קובץ ההגדרה הראשי של GRUB 1.
cat /boot/grub/menu.lst מציג את תוכן קובץ ההגדרה.
grub מפעיל את ממשק GRUB Legacy (command line).
find /boot/vmlinuz פקודת GRUB – מחפש את קובץ הקרנל.
root (hd0,0) פקודת GRUB – מגדיר partition ראשי.
kernel /boot/vmlinuz root=/dev/sda1 פקודת GRUB – טוען קרנל עם פרמטרים.

GRUB version 2

GRUB 2 – הגרסה המודרנית. קובץ ההגדרה grub.cfg נוצר אוטומטית מ-קבצים ב-/etc/grub.d/.

/boot/grub/grub.cfg קובץ ההגדרה הראשי של GRUB 2 (לא לערוך ישירות).
/etc/grub.d/ תיקיית סקריפטים ליצירת grub.cfg.
/etc/default/grub קובץ הגדרות ברירת מחדל ל-GRUB 2.
update-grub מעדכן את grub.cfg (Debian/Ubuntu).
grub2-mkconfig -o /boot/grub2/grub.cfg מעדכן את grub.cfg (Fedora/RHEL עם BIOS).
grub2-mkconfig -o /boot/efi/EFI/fedora/grub.cfg מעדכן את grub.cfg (Fedora/RHEL עם UEFI).
grub2-editenv list מציג משתני סביבה של GRUB 2.
grub2-set-default 0 מגדיר את ברירת המחדל לאתחול (0=הראשון).

Rescue boot

תצורת אתחול חירום – מאפשרת תיקון מערכת כשהאתחול הרגיל נכשל.

init=/bin/bash פרמטר קרנל – מאתחל ישירות ל-bash במקום init.
init=/bin/sh מאתחל ל-shell בסיסי.
single פרמטר קרנל – מאתחל ל-mode single-user.
systemctl rescue מעבר ל-rescue mode (מערכת מינימלית).
systemctl emergency מעבר ל-emergency mode (מינימלי עוד יותר).
chroot /mnt/sysimage משנה root ל-partition של המערכת (מתקן דיסק חי).
mount -o remount,rw / מעמיד מחדש את / לקריאה-כתיבה.

Summary

נושא תיאור
תהליך האתחול BIOS/UEFI → Bootloader (GRUB) → Kernel → init/systemd
GRUB 1 קובץ הגדרה: /boot/grub/menu.lst
GRUB 2 grub.cfg נוצר מ-/etc/grub.d/ ו-/etc/default/grub
אתחול חירום init=/bin/bash, single, systemctl rescue/emergency
