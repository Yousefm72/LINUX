Introduction to File Systems

מערכת קבצים (file system) מגדירה איך נתונים מאוחסנים ומאורגנים על דיסק. היא מנהלת קבצים, תיקיות, הרשאות ומטא-דאטה (inodes, blocks).

מערכות קבצים נפוצות ב-Linux: ext4 (ברירת מחדל), XFS (לקבצים גדולים), btrfs (snapshots), FAT32/exFAT (התקנים ניידים).

ext4 מערכת קבצים ראשית ב-Linux. יציבה, journaling.
XFS מתאימה לקבצים גדולים. הרחבה בלבד (אין הקטנה).
btrfs תמיכה ב-snapshots, RAID תוכנתי, checksums.
inode מבנה מטא-דאטה לקבצים – הרשאות, גודל, מצביעים לבלוקים.

Creating File Systems

יצירת מערכת קבצים על מחיצה או LV עם mkfs או הפקודות הספציפיות. אזהרה: פעולה זו מוחקת את כל הנתונים על ההתקן.

mkfs.ext4 /dev/sda1 יוצר מערכת קבצים ext4 על המחיצה.
mkfs.xfs /dev/sda2 יוצר מערכת קבצים XFS.
mkfs -t ext4 /dev/sda1 צורה חלופית – mkfs עם אופציה -t.
mkfs.vfat /dev/sdb1 יוצר FAT32 (להתקנים ניידים).

Creating Partitions and SWAP Files

מחיצות נוצרות עם fdisk או parted. SWAP – זיכרון וירטואלי על דיסק – יכול להיות מחיצה ייעודית או קובץ swap.

fdisk /dev/sdb עורך מחיצות – n (חדש), t (סוג), w (שמור).
parted /dev/sdb עורך מחיצות – תומך ב-GPT ובגדלים מעל 2TB.
mkswap /dev/sdb2 מגדיר מחיצה כ-swap.
swapon /dev/sdb2 מפעיל swap (טמפוררי).
dd if=/dev/zero of=/swapfile bs=1M count=1024 יוצר קובץ swap בגודל 1GB.
chmod 600 /swapfile && mkswap /swapfile הגדרת הרשאות ויצירת swap מקובץ.
swapon /swapfile מפעיל swap מקובץ.
swapoff /dev/sdb2 מפסיק swap.

Mounting File Systems – Temporarily

mount זמני – מחלקת ה-mount נעלמת אחרי הפעלה מחדש. משתמשים ב-mount ו-umount.

mount /dev/sda1 /mnt מעמיד מחיצה ב-/mnt.
mount -t ext4 /dev/sda1 /mnt/data מעמיד עם סוג קבצים מפורש.
mount -o ro /dev/sda1 /mnt מעמיד לקריאה בלבד (read-only).
umount /mnt מנתק מחיצה. או: umount /dev/sda1
mount מציג את כל ה-mounts הפעילים.

Mounting File Systems – Permanently

mount קבוע – רשומה ב-/etc/fstab. המערכת מעמידה אוטומטית באתחול. כל שורה: device mount_point fstype options dump fsck

Example /etc/fstab entry:

UUID=xxx  /mnt/data  ext4  defaults  0  2
/etc/fstab קובץ הגדרות mount אוטומטי.
blkid מציג UUID – עדיף ל-fstab מאשר /dev/sda1 (יציב יותר).
mount -a מעמיד את כל השורות מ-fstab שלא מאוישות.
swap /swapfile none swap sw 0 0 שורת fstab ל-swap מקובץ.

סיכום ותרגילים בעברית עם דוגמאות

מערכות קבצים: ext4, XFS, btrfs. יצירה עם mkfs.ext4, mkfs.xfs.

מחיצות ו-swap: fdisk/parted ליצירת מחיצות. mkswap ו-swapon ל-swap.

Mount: זמני – mount/umount. קבוע – /etc/fstab עם UUID.

תרגיל 1 – מחיצה וקובץ swap:

  • dd if=/dev/zero of=/swapfile bs=1M count=512 – יצירת קובץ 512MB
  • chmod 600 /swapfile && mkswap /swapfile && swapon /swapfile – הפעלת swap

תרגיל 2 – mount קבוע:

  • blkid /dev/sdb1 – קבלת UUID
  • הוספת שורה ל-fstab: UUID=xxx /mnt/backup ext4 defaults 0 2
  • mount -a – בדיקה
