Packages on Linux

חבילות (Packages) הן יחידות התקנה של תוכנה ב-Linux. כל חבילה מכילה קבצים, תלויות (dependencies) ומטא-דאטה. מנהלי חבילות מטפלים בהתקנה, עדכון והסרה.

שתי משפחות עיקריות: Debian-based (Ubuntu, Debian) משתמשת ב-.deb ו-apt; RHEL-based (CentOS, Fedora, RHEL) משתמשת ב-.rpm ו-yum או dnf.

DPKG – כלי ברמה נמוכה לניהול חבילות .deb. apt משתמש ב-dpkg מתחת למכסה.

RPM – כלי ברמה נמוכה לניהול חבילות .rpm. yum ו-dnf משתמשים ב-rpm מתחת למכסה.

.deb פורמט חבילה של Debian/Ubuntu.
.rpm פורמט חבילה של Red Hat/CentOS/Fedora.
/var/lib/dpkg/ מסד נתונים של חבילות מותקנות (Debian).
/var/lib/rpm/ מסד נתונים של חבילות מותקנות (RPM).
dpkg -i package.deb מתקין חבילת .deb.
dpkg -r packagename מסיר חבילה אך משאיר קבצי הגדרה.
dpkg -P packagename מסיר חבילה יחד עם קבצי הגדרה (purge).
dpkg -l מציג את כל החבילות המותקנות.
dpkg -l | wc -l ספירת מספר החבילות המותקנות.
dpkg -l | grep mysql מחפש חבילות מותקנות שמכילות "mysql" בשם.
dpkg -S ifconfig מציג לאיזו חבילה שייך הקובץ ifconfig.
dpkg -c package.deb מציג את תוכן חבילת .deb (רשימת קבצים) בלי להתקין.
rpm -i package.rpm מתקין חבילת .rpm.
rpm -e packagename מסיר (uninstall) חבילה.
rpm -q packagename בודק אם חבילה מותקנת ומציג את הגרסה.
rpm -qa מציג את כל החבילות המותקנות.
rpm -qa | wc -l ספירת מספר החבילות המותקנות.
rpm -qa | grep mysql מחפש חבילות מותקנות שמכילות "mysql" בשם.
rpm -qf /usr/sbin/ifconfig מציג לאיזו חבילה שייך הקובץ (query file).
rpm -ql net-tools מציג את רשימת הקבצים בחבילה net-tools (query list).

Using apt on Ubuntu

apt (Advanced Package Tool) הוא מנהל החבילות ב-Ubuntu ו-Debian. מטפל בתלויות אוטומטית ומתחבר ל-repositories מרוחקים.

apt update מרענן את רשימת החבילות הזמינות (מטא-דאטה).
apt upgrade מעדכן את כל החבילות המותקנות.
apt install packagename מתקין חבילה. דוגמה: apt install nginx
apt remove packagename מסיר חבילה אך משאיר קבצי הגדרה.
apt purge packagename מסיר חבילה יחד עם קבצי הגדרה.
apt search keyword מחפש חבילות לפי מילת מפתח.
apt show packagename מציג מידע מפורט על חבילה.
apt list --installed מציג את כל החבילות המותקנות.
apt autoremove מסיר חבילות תלויות שלא בשימוש.

Repositories: /etc/apt/sources.list – רשימת מקורות החבילות.

Using yum on CentOS

yum (Yellowdog Updater Modified) הוא מנהל החבילות ב-CentOS ו-RHEL הישנים. ב-Fedora ו-RHEL 8+ הוחלף ב-dnf, אך הפקודות דומות.

yum update מעדכן את כל החבילות (משלב refresh והתקנה).
yum install packagename מתקין חבילה. דוגמה: yum install httpd
yum remove packagename מסיר חבילה.
yum search keyword מחפש חבילות לפי מילת מפתח.
yum info packagename מציג מידע מפורט על חבילה.
yum list installed מציג את כל החבילות המותקנות.
yum repolist מציג את ה-repositories הפעילים.
dnf install packagename גרסה מודרנית – dnf (Fedora, RHEL 8+).

Repositories: /etc/yum.repos.d/ – קבצי .repo מגדירים את המקורות.

סיכום בעברית עם דוגמאות

חבילות ב-Linux: Debian/Ubuntu משתמשים ב-.deb ו-apt; CentOS/RHEL משתמשים ב-.rpm ו-yum/dnf.

apt (Ubuntu):

  • apt update – רענון רשימת חבילות
  • apt install nginx – התקנת nginx
  • apt remove nginx – הסרת חבילה
  • apt search vim – חיפוש חבילות

yum (CentOS):

  • yum install httpd – התקנת Apache
  • yum update – עדכון כל החבילות
  • yum remove httpd – הסרת חבילה
  • yum info nginx – מידע על חבילה

טיפ: תמיד להריץ apt update לפני apt install; ב-yum הפקודה update מטפלת בזה אוטומטית.
