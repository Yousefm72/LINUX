Introduction to IP Networking

רשת IP משתמשת בפרוטוקול TCP/IP לתקשורת בין מחשבים. כל מחשב מזוהה לפי כתובת IP ייחודית. IPv4 (32 bit, דוגמה: 192.168.1.10) ו-IPv6 (128 bit) – שתי גרסאות הנמצאות בשימוש.

רכיבים עיקריים: IP address (מזהה המחשב), subnet mask (גודל הרשת), gateway (נתב יציאה), DNS (תרגום שמות לכתובות).

192.168.1.0/24 רשת מקומית – 24 bits לרשת, 256 כתובות.
gateway נתב – מעביר חבילות לרשתות חיצוניות.
DNS Domain Name System – תרגום שם לכתובת IP.

IP Addressing and Configuration

הגדרת IP ב-Linux מתבצעת עם ip (מודרני) או ifconfig (ישן). קבצי הגדרה: /etc/network/interfaces (Debian) או /etc/sysconfig/network-scripts/ (RHEL).

ip addr show מציג את כל הממשקים והכתובות.
ip addr add 192.168.1.10/24 dev eth0 מוסיף כתובת IP לממשק (זמני).
ip link set eth0 up מפעיל ממשק רשת.
ifconfig מציג ממשקים (פקודה ישנה, net-tools).

Network Interfaces

ממשק רשת (network interface) – eth0, ens33, enp0s3 – מתאם הרשת. ip link מציג ומנהל ממשקים.

ip link show מציג את כל ממשקי הרשת.
ip link set eth0 down משבית ממשק.
ethtool eth0 מידע על ממשק (מהירות, קישור).
/sys/class/net/ מידע על ממשקים ב-sysfs.

Static vs Dynamic IP (DHCP)

Static IP – כתובת קבועה, מוגדרת ידנית. מתאים לשרתים. Dynamic IP (DHCP) – הכתובת מתקבלת אוטומטית משרת DHCP.

dhclient eth0 מבקש כתובת IP משרת DHCP.
nmcli device status מציג סטטוס (עם NetworkManager).
nmcli connection add type ethernet con-name mynet ifname eth0 ipv4.addresses 192.168.1.10/24 ipv4.gateway 192.168.1.1 יצירת חיבור סטטי עם NetworkManager.
/etc/resolv.conf הגדרות DNS (nameservers).

Routing and Connectivity

בדיקת קישוריות ו-routing – ping, traceroute, ip route, ss, netstat.

ping 8.8.8.8 בודק קישוריות לכתובת (ICMP).
traceroute google.com מציג את מסלול החבילות ליעד.
ip route show מציג טבלת routing.
ip route add default via 192.168.1.1 מוסיף gateway ברירת מחדל.
ss -tuln מציג listening ports (TCP/UDP).
hostname -I מציג את כתובות ה-IP של המכונה.

סיכום ותרגילים בעברית עם דוגמאות

IP networking: כתובת IP, subnet, gateway, DNS. IPv4 ו-IPv6.

הגדרה: ip addr, ip link. קבצים: /etc/network/interfaces או NetworkManager.

פקודות עיקריות:

  • ip addr show – הצגת כתובות
  • ping 8.8.8.8 – בדיקת קישוריות
  • ip route show – טבלת routing

תרגיל 1 – הגדרת IP זמנית:

  • ip addr add 192.168.1.100/24 dev eth0
  • ip route add default via 192.168.1.1

תרגיל 2 – בדיקת רשת:

  • ping -c 3 192.168.1.1 – בדיקת gateway
  • ss -tuln – רשימת פורטים פתוחים
