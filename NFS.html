NFS Principles

NFS (Network File System) מאפשר שיתוף תיקיות בין מחשבים ברשת – שרת מעמיד (export) תיקיות, ולקוחות מעמידים אותן כ-mount point מקומי. הגישה שקופה – המשתמש מרגיש כאילו הקבצים מקומיים.

NFS משתמש ב-RPC (Remote Procedure Call). השרת רץ nfsd; הלקוח משתמש ב-mount.nfs. /etc/exports מגדיר אילו תיקיות משותפות ולמי.

export הפעלת שיתוף תיקייה מהשרת לרשת.
RPC Remote Procedure Call – פרוטוקול התקשורת של NFS.
/etc/exports קובץ ההגדרות – תיקיות משותפות ורשימת clients מורשים.

NFS Versions

NFS v2 (ישן), NFS v3 (נפוץ), NFS v4 (מודרני, ברירת מחדל). NFSv4 עובד על פורט 2049 בלבד, תומך ב-firewall, ונקודות mount מרובות (pseudo-filesystem).

NFSv3 נתמך בכל מקום. UDP/TCP. stateless.
NFSv4 פורט 2049, stateful, אבטחה משופרת, ACLs.
port 2049 פורט NFSv4 – פשוט יותר ל-firewall מ-NFSv3.

NFS Server Administration

ניהול שרת NFS – התקנת חבילות, עריכת /etc/exports, הפעלת שירותים. Debian: nfs-kernel-server; RHEL: nfs-utils.

Example /etc/exports:

/export/shared  192.168.1.0/24(ro,sync)
/home/data      192.168.1.10(rw,sync,no_subtree_check)
exportfs -a מייצא את כל התיקיות מ-exports.
exportfs -ra טוען מחדש exports (re-export) אחרי שינוי בקובץ.
exportfs -v מציג את ה-exports הפעילים.
showmount -e server_ip מציג תיקיות שמוצאות משרת NFS.
systemctl start nfs-server מפעיל שרת NFS (systemd).
ro / rw read-only או read-write ב-exports.

NFS Client-Side Administration

צד הלקוח – התקנת nfs-common (Debian) או nfs-utils (RHEL), mount של תיקיות משותפות מהשרת.

showmount -e 192.168.1.1 מציג exports זמינים מהשרת.
mount -t nfs 192.168.1.1:/export/shared /mnt/nfs מעמיד NFS share מקומית (זמני).
mount 192.168.1.1:/home/data /mnt/data ב-NFSv4 -t nfs אופציונלי.
umount /mnt/nfs מנתק mount. אם תפוס: umount -l (lazy).
/etc/fstab: server:/export /mnt nfs defaults 0 0 mount קבוע באתחול.

Using AutoFS

AutoFS מעמיד NFS (או התקנים אחרים) אוטומטית רק כשנכנסים לתיקייה – חוסך זמן אתחול ומונע בעיות כשהשרת לא זמין. מוגדר ב-/etc/auto.master ו-/etc/auto.nfs.

autofs שירות – mount on-demand.
/etc/auto.master מגדיר mount points ו-maps.
/etc/auto.nfs מפת NFS – שרת:path ל-mount.
/+ auto.nfs שורה ב-auto.master – /net או /mnt/autofs.
systemctl enable --now autofs מפעיל AutoFS באתחול.

Example /etc/auto.nfs:

data  -rw,bg  192.168.1.1:/export/shared

סיכום ותרגילים בעברית עם דוגמאות

NFS: שיתוף תיקיות ברשת. שרת – exports; לקוח – mount.

גרסאות: NFSv3, NFSv4 (פורט 2049).

פקודות שרת:

  • exportfs -ra – רענון exports
  • showmount -e localhost – הצגת exports

פקודות לקוח:

  • mount server:/export /mnt – mount NFS

תרגיל 1 – שרת:

  • הוסף ל-exports: /srv/nfs *(ro,sync)
  • exportfs -ra

תרגיל 2 – לקוח:

  • showmount -e 192.168.1.1
  • mount 192.168.1.1:/srv/nfs /mnt/nfs
